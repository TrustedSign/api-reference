# 获取令牌

令牌token用于页面集成方式时的权限验证。

方法：`POST: /token`

说明：token有效期为1hour, 可续订token

| 名称        | 类型        | 说明  |
| ------------- |-------------| -----|
| `token` | String	 | 令牌，续订用  |

响应数据列表:

| 名称        | 类型        | 说明  |
| ------------- |-------------| -----|
| `token` | String	 |   身份令牌，续订失败会返回新的token    |
