## 平台简介
[可信签](https://trustedsign.com)项目是一个基于SaaS模式的文档签名的云平台，使用模式包括个人对个人、企业对企业、企业对个人等。实名认证后的个人和企业可使用颁发的证书对文件进行签名、流转、存档等。平台提供开发接口，可与其他系统实现集成。

## 阅读对象
本文档阅读对象：涉及集成可信签电子合同生成、签名及其他相关操作的技术架构师，研发工程师，测试工程师，系统运维工程师等。
